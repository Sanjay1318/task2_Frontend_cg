# Login & Registration Pages

A simple front-end assignment built with HTML and CSS.

## Pages
- **login.html** - User login form with username and password fields
- **registration.html** - User registration form with fields for username, full name, email, date of birth, education, address, phone number, gender, and password

## Features
- External CSS (`external_new.css`) for styling both pages
- Different background images for login and registration pages
- Gradient submit buttons
- Navigation links between login and registration pages
- Redirects to the respective page on form submit

## How to Run
Open `login.html` or `registration.html` in a browser using Live Server or any browser directly.
